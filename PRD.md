# 资源中心系统需求文档 (PRD)

## 1. 项目概述
本项目旨在构建一个企业级的资源中心系统，支持资源的统一发现（资源广场）与精细化权限配置（资源管理）。系统提供多种资源类型（如模型、知识库、数据源等）的展示、搜索及授权管理功能。

## 2. 核心功能模块

### 2.1 资源广场 (Resource Plaza)
用户发现和使用资源的入口。
*   **多维度分类**：支持模型、知识库、工具、工作流、数据源等多种类型的 Tab 切换。
*   **搜索与过滤**：
    *   搜索框集成于 Tab 栏右侧（右对齐），支持按名称、标签或 ID 搜索。
*   **资源卡片个性化展示**：
    *   **通用信息**：资源名称、所有者、描述、可见性、创建日期。
    *   **个性化标签**：
        *   **模型**：展示模型类型（大模型、rerank、向量化模型）及部署方式（公网、本地）。
        *   **知识库**：展示知识库类型（文档知识库、结构化知识库、图结构知识库）。
        *   **数据源**：展示数据库类型（MySQL/TiDB/MinIO）。

### 2.2 资源管理 (Resource Management)
资源所有者或管理员管理其拥有或可管理的资源。
*   **列表展示**：卡片式展示资源概览。
*   **权限配置入口**：点击卡片进入“权限管理”抽屉。

### 2.3 权限管理抽屉 (Permission Management Drawer)
提供精细化的权限管控功能。

#### 2.3.1 授权对象列表
*   **人员信息展示**：列表展示人员姓名及其所属部门。
*   **所有权转移**：对于“所有权人”，在操作列提供“所有权转移”按钮，点击弹出人员选择弹窗。
*   **添加授权对象**：
    *   点击“添加授权对象”按钮，弹出对象选择弹窗。
    *   支持批量选择人员，提供内部搜索功能。
*   **批量操作**：
    *   **批量修改权限**：弹出模态框，下拉选择权限级别（可见、使用、管理）。
    *   **批量设置到期日期**：弹出模态框，可选择日期或勾选“永久有效”。
    *   **批量移除**：点击后展示二次确认弹窗，确认后移除选中的授权对象。

#### 2.3.2 审批列表
*   展示待处理的资源使用申请。
*   支持查看申请详情、批准或驳回（需填写驳回原因）。

#### 2.3.3 全局公开策略设置
*   支持设置资源的公开范围：私有、公开可见（需申请）、公开可用。
*   **保存机制**：操作按钮位于抽屉右下角页脚处，文案为“保存”。

## 3. UI/UX 规范
*   **动画效果**：使用 `motion` 实现流畅的页面与弹窗过渡。
*   **组件库**：基于 Ant Design 5.0 风格进行定制化开发。

## 4. 技术方案
*   **前端框架**：React 18 + TypeScript + Vite
*   **样式处理**：Tailwind CSS + Ant Design
*   **状态管理**：React Hooks / Context API
*   **图标库**：Lucide React + Ant Design Icons
